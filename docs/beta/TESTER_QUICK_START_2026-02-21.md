# SkateFlow Beta Tester Quick Start

Date: Saturday, February 21, 2026
Session length: 20-30 minutes

## 1) Open the app
Use this URL on your device browser:
- Beta URL: `http://localhost:5173/`

Note: `localhost` works only on the same machine running the app. For phones/tablets, use your computer's LAN IP (example: `http://192.168.1.23:5173/`).

If it does not load:
- Make sure you are on the same Wi-Fi as the host machine.
- Re-open the link in Chrome/Safari.

## 2) Login
- Choose your user
- Enter PIN

## 3) Run this quick flow (in order)
1. Log tab
- Add landed/missed reps
- Upload 1 photo + 1 video
- Trim the video (tap `Trim`)
- Save Session Card

2. Cards tab
- Open latest session card
- Confirm media displays
- Edit the card and save

3. Contest tab
- Add run (name/date/round/run #)
- Add 2 tricks and mark landed/missed
- Upload run photo/video
- Reorder runs

4. Coach tab
- Upload a coach demo video for current skater

5. Stats tab
- Use Park Conditions search
- Save a park
- Refresh weather

6. Settings tab
- Switch Light/Dark mode
- Export backup JSON
- Run Health Check

## 4) What to report immediately
Report any of these right away:
- White screen / crash
- Save button not working
- Media disappears after save
- Data resets unexpectedly
- Login/PIN fails incorrectly
